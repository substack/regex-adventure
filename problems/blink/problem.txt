Створiть аналог методу markdown, який переводиіть весь текст мiж `@@...@@` в текст `<blink>...</blink>`.

Наприклад, `@@whatever@@` замiняється на `<blink>whatever</blink>`.

Звернiть увагу на те, що markdown дозволяє використовувати конструкцiї `@@...@@` всерединi `@@...@@`. Ви повиннi реалiзувати цю можливiсть.

Ваш код повинен прийняти строку з розмiткою в синтаксисi markdown та повернути строку в html. Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    /* return html string */
  }

Для цього завдання використовуйте `marked` модуль. Щоб його встановити виконайте команду: `npm install marked`

Для того щоб виконати замiну спробуйте метод `.replace()`
