Зараз треба створити модуль, що виконує пошук чісла яке йде за пiдстрокою `x=`, та повертає знайдене чiсло.

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    var m = /...your regex goes here.../.exec(str)
    return m ? m[1] : null
  }

Замiсть визову метода `.test()` ми маємо робити визов методу `.exec()`, який повертає знайдений результат. Якщо регулярний вираз спiвпадає зi строкою, то `.exec()` повертає match захваченої групи.

Для того щоб створити групу для захвата в регулярному виразi ви повиннi додати на початку на кiнцi групи круглi дужки `(` та `)`.

Текст, що розташовано мiж такими дужками буде захвачено та повернено в якостi резутату виконання методу `.exec()` у виглядi масиву з єлементами починаючи з крайнього лiвого (якщо таких спiвподiнь два або бiльше).
