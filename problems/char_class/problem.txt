Створiть модуль, що повертає true в разi, якщо строка починається з малої голосної латинської лiтери або цифри.

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    /* return whether str starts with a vowel or a digit */
  }

Для проходження цього етапу вам знадобляться пiдмножини.

Символи, що росташованi мiж вiдкриваючими та закриваючими квадратними дужками називають пiдмножина `[...]`. Кожен символ такої пiдмножини буде перевiрятися на спiвпадiння з символами строки для тестування.
Наприклад, `[abcdexyz]` дає спiвпадiння для кожного символу окремо `a`, `b`, `c`, `d`, `e`, `x`, `y`, чи `z`. Однак, той самий вираз можна записати використовуючи дiапазон символiв (роздiлених знаком `-`): 
наприклад `[a-exyz]`

Чи ви ще пам'ятаєте, що регулярнi вирази починаются, та закiнчуються символами `/`, а для того, щоб протестувати вираз на вiдповiднiсть до строки, використовують методод `.test()`?
