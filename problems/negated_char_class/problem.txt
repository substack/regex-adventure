Створiть модуль, що повертає true в разi, якщо перший символ строки не цифра, а другий символ строки не велика лiтера.

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    // ...
  }

Для того, щоб вдало пройти цей рiвень вам треба навчитися використовувати заперечення в регулярних виразах.
Заперечення позначається знаком `^` на початку пiдмножини, внутрi квадраних дужок `[]`.

Нагадаємо, що символ `^` за межами квадратних дужок позначає початок строки. Це важливо, будьте уважні.

Для того, щоб скоротити записи регулярних виразiв можна використовувати метасимволи.


  * \d - [0-9]        будь-яка цифра
  * \w - [A-Za-z_]    будь-яка лiтера
  * \s - [ \t\r\n\f]  пробіл

Великi символи позначають протележне значення:

  * \D - [^0-9]        не цифра
  * \W - [^A-Za-z_]    не лiтера
  * \S - [^ \t\r\n\f]  не пробіл

Спецiальний метасимвол `.` означає будь-який символ, за виключенням переносу строки.

Щоб перевiрити ваше рiшення:

  regex-adventure-ua verify solution.js
