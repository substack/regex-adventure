Створiть модуль, що отримує строку, та повертає масив зi строк, що були заключенi у подвiйнi лапки. Вихiдний масив має мiстити строки разом iз лапками.


Наприклад, вхiдна строка: 'cool "beans" "beep boop" whatever "yay"'
ваша програма повинна повернуті масив: ['"beans"', '"beep boop"', '"yay"']

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    return str.match(/.../g)
  }

Ви повиннi використовувати `str.match(regex)` щоб повернути массив для `regex` з флагом `/g`.

Вам можуть знадобитися заперечення в регулярних виразах.
