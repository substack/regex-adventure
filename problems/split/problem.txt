Строки в javascript мають метод `.split()`. Цей метод роздiляє строку на частини, та результат повертає у виглядi масиву. Цей метод може приймати в якостi роздiльника строку або регулярний вираз.

Сворiть модуль, що приймає строку, та повертає масив, елементами якого є частини строки, що були роздiленi символамі ` , `(кома з пробiлами на кiнцях)

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    return str.split(/* ... your regex goes here ... */)
  }

Нагадаємо, що `\s` – це скорочення для символу пробiлу `[ \t\r\n\f]`.
