# regex-adventure-ua

Вивчайте регулярнi вирази разом з воркшопом.

# Встановлення

```
$ sudo npm install -g regex-adventure-ua
```

Вiдтепер у вас доступна `regex-adventure-ua` команда:

```
$ regex-adventure-ua
```

Щоб проходити цей воркшоп оффлайн, додайте:

```
$ npm install quotemeta marked
```
